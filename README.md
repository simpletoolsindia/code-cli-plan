## Research Summary

**Repositories Analyzed:**
- Claude Code (TypeScript) - Best tool factory, compaction
- Codex-RS (Rust) - Best sandbox, TUI, ghost commits
- Aider (Python) - Best git integration, RepoMap, lazy loading
- Cline (TypeScript) - Best MCP hub, 40+ providers
- OpenCode (TypeScript) - Best LSP, Effect-based DI

**Key Features to Implement:**
- Ghost Commits (Codex-RS)
- RepoMap PageRank (Aider)
- 6-Flag Git Attribution (Aider)
- McpHub 59KB (Cline)
- Memory Taxonomy (Claude Code)
- 50K Token Compaction (Claude Code)
- Landlock Sandbox (Codex-RS)
- 28+ LSP Servers (OpenCode)
